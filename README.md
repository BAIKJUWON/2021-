
> # __Team 고속버스의 작품 Doge 화분__
> 
# 소개
**Doge 화분**은 현대인의 바쁜 생활 속에서 식물 관리의 부담을 덜어주는 혁신적인 스마트 화분입니다.  
친환경적인 소재와 스마트 기술을 결합하여, 식물 관리가 필요 없는 새로운 개념의 화분을 제공합니다.  
탄소 중립을 위한 환경 보호와 사용자 편의를 모두 고려한 설계로, 일상에 자연의 아름다움을 더합니다.
